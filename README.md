# rec-cx-combin

## Install

```bash
bun install
```

## Dev Run

```bash
bun run dev
```

## One-Click Build (Windows x64)

```bash
bun run build:win:oneclick
```

Output package path:

`dist/oneclick-win-x64`

Inside the package:
- `rec-cx-combine.exe`
- `run.bat` (double-click to run)
- `input/` (put source files here)
- `output/` (generated file path)
